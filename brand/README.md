# CableTwin brand kit

- Name: **CableTwin** — Signature: **The Waze of the cable factory**
- Brand idea: three cable lines; one stops (orange); a mint recovery route
  bypasses the stop — the product story in one mark.
- Palette: deep green `#092925`, teal `#0F766E`, mint `#C9F36B`, cream
  `#F5F4ED`, white `#FFFFFF`, orange accent `#E9763B` (incident only)
- Typography: Segoe UI / Arial (standard portable fonts only)

## Contents

```text
logo/
  cabletwin-logo-light.svg   for light backgrounds (+ cabletwin-logo.png)
  cabletwin-logo-dark.svg    for dark backgrounds  (+ cabletwin-logo-dark.png)
  cabletwin-logo-monochrome.svg
  cabletwin-mark.svg         compact mark (+ cabletwin-mark.png)
social/
  cabletwin-demo-thumbnail.png   1280x720 video thumbnail (+ thumbnail.html source)
CableTwin_Brand_Guide.pdf
```

Regenerate PNG exports with `node scripts/render-brand.mjs`; regenerate the
guide with `python scripts/build_brand_guide.py`.

Rule: no visual association with any real cable manufacturer; all operational
results shown near the brand are labelled synthetic.

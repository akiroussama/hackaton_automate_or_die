# Mandatory Phase 2 coordination

Before any repository action, read:

`docs/collaboration/phase2-codex-claude-handoff.md`

Only the active owner named in that file may work. The other agent must stop.
Every block ends with a written handoff containing files, commands, checks,
risks, next owner and next permitted action.

The user must not be used as a message relay. Claude Code and Codex communicate
through the shared handoff file.
